# 探店打卡类钩子

> 适用场景：咖啡店、餐厅、展览、景点、打卡分享
> 指令层级：L1-收藏 → L2-转发 → L3-行动
> **合规声明**：探店类钩子不涉及商品功效承诺，合规风险较低，但需注意避免"最好/第一"等表述。
