# 正文结构模板 — /redbook

> 模板是起点，不是终点。填你自己的内容、你自己的故事、你自己的语气。
> 所有模板中的 {花括号} 内容必须来自用户 Fact Sheet，不自动补全。

## 通用四段式

适用于所有类型的小红书笔记。

```
段1：开头钩子（1-3句）
  → 痛点/悬念/共鸣/惊叹
  → 必含：至少1个称呼（姐妹/宝子/家人们）
  → 目标：3句内让读者停下来

段2：故事/场景（3-6句）
  → 个人经历/痛点展开/场景描写
  → 必含：至少1个称呼 + 1个emoji
  → 目标：让读者觉得"说的就是我"

段3：干货/价值（3-8句）
  → 步骤/清单/对比/推荐
  → 必含：每3-4句1个emoji
  → 目标：让读者觉得"学到了"或"想要"

段4：结尾钩子（最后3句）
  → 倒数第3句：钩子注入（从 hooks/{type}.md 选取）
  → 倒数第2句：情感收束/金句
  → 倒数第1句：自然结束（可含称呼或emoji）

**语域密度检查点**：每段至少1个称呼或emoji，全文称呼密度≥0.2，emoji密度0.2-0.3。
```

### 情感故事模板 — 白描风格（V1.6 新增）

> **适用场景**：用户偏好平静白描、不端着、有毛边、少emoji、不要亲昵称呼
> **核心差异**：去掉"姐妹们/宝子们"等称呼，去掉煽情元素，用白描手法讲述真实故事

```
【开头钩子】
{时间}，{场景}。

【经历】
📌 {关键事件}。
{具体细节——白描手法：不加形容词，只写事实}。
📌 {关键对话——原话，不润色}。
📌 {转折——一个瞬间，不需要解释为什么}。

【感悟】
{一句话感悟——短句，不展开}。

【结尾钩子】
{hooks/emotion.md L1-收藏级钩子}
```

**白描风格规则**：
- 称呼：去掉所有称呼（"姐妹们/宝子们"→无）
- emoji：去掉所有 emoji（💛→无）
- 语气词：去掉"真的/绝了/太XX了"等情绪词
- 句式：短句为主（≤15字），长句≤1句/段
- 感官：只写事实，不加感受（"窗外下着雨"而非"窗外下着雨，让人伤感"）
- 对话：原话引用，不润色（"他说：'算了'"而非"他叹了口气说：'算了'"）
- 结尾：留白，不总结（"就这样吧"而非"这就是成长的意义"）

## 对比测评模板

**结构**：困惑 → 对比 → 结论 → 钩子

```
【开头钩子】
📌 {A}和📌 {B}到底选哪个？我帮你试了！

【对比】
| 维度 | 📌 {A} | 📌 {B} |
|------|--------|--------|
| 价格 | 📌 {价格} | 📌 {价格} |
| {维度2} | 📌 {评价} | 📌 {评价} |
| {维度3} | 📌 {评价} | 📌 {评价} |

【结论】
✨ {一句话结论——谁适合A，谁适合B}
💡 {口语化补充——"我个人觉得""你要是XX的话建议选""说实话区别不大"}

【结尾钩子】
{hooks/compare.md 钩子}
```

**语域铁律**：对比测评必须有≥2句口语化主观判断（"说实话""我实测下来""你要是XX就选A"），不能纯列数据。
