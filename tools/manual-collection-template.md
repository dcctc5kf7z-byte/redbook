# 小红书爆款笔记手动采集模板

> **用途**：手动采集同类目 TOP 20 爆款笔记数据，建立初始爆款基准线
> **采集方法**：小红书 App/Web → 搜索关键词 → 按「最热」排序 → 记录前 20 条
> **采集频率**：每个类目 1 次，每月更新 1 次

## 采集完成后

1. **计算基准线**：每个类目的平均赞/藏/评 → 写入 `viral-patterns.jsonl`
2. **分析标题公式**：TOP 20 标题用了哪些公式 → 更新标题模板
3. **分析钩子类型**：TOP 20 开头用了什么钩子 → 更新钩子库
4. **分析标签**：TOP 20 高频标签 → 更新标签策略

### 基准线计算公式

```
高互动阈值 = 平均值 + 1 个标准差
中位数阈值 = 排序后第 10 条的值
低互动阈值 = 平均值 - 1 个标准差
```

### 转换为 JSONL 格式

采集完成后，将每条数据转换为 JSONL 格式追加到 `data/viral-patterns.jsonl`：

```json
{
  "date": "2026-07-01",
  "content_type": "好物",
  "title_type": "数字清单",
  "title": "5个平价面霜推荐，第3个我后悔没早买",
  "likes": 5234,
  "collects": 2891,
  "comments": 467,
  "tags": ["面霜", "平价", "护肤"],
  "source": "manual_top20",
  "schema_version": "2.0"
}
```
