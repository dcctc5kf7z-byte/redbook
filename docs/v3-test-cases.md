## 详细用例

### 用例 1：seeder × 好物推荐（内容类型默认）

**输入**：
```
推荐珀莱雅双抗精华，用了3个月，皮肤真的变好了，价格不到200
```

**预期路由**：content_type=好物推荐 → persona=seeder
**预期称呼**：姐妹们/宝子们
**预期语气**：热情、有感染力、有购买冲动
**预期 emoji**：0.15-0.25（🔥💰✅）
**验收点**：称呼是"姐妹们"而非"各位"；有"闭眼入/冲"类口头禅；不像广告词

---

### 用例 2：lecturer × 教程攻略（内容类型默认）

**输入**：
```
教你3步学会拍日系写真，手机就能拍，不需要相机
```

**预期路由**：content_type=教程攻略 → persona=lecturer
**预期称呼**：同学们/各位
**预期语气**：沉稳、清晰、有条理
**预期 emoji**：0-0.05
**验收点**：称呼是"同学们"而非"姐妹们"；有"划重点/注意了"类口头禅；逻辑清晰不煽情

---

### 用例 3：bff × 情感共鸣（内容类型默认）

**输入**：
```
分手后第30天，我终于想通了
```

**预期路由**：content_type=情感共鸣 → persona=bff
**预期称呼**：姐妹们/宝子
**预期语气**：亲密、有分享欲
**预期 emoji**：0.25-0.35
**验收点**：白描模式触发（动作+对话为主）；不像"感动中国"；有"谁懂啊"类口头禅

### 用例 4：aesthete × 探店打卡（内容类型默认）

**输入**：
```

```

**预期路由**：content_type=探店打卡 → persona=aesthete
**预期称呼**：喜欢XX的你
**预期语气**：温柔、细腻、有画面感
