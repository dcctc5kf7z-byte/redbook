## 三、《上线前必须加固的 6 项防御性工程改造清单》

### 改造 #1（P0）：Fact Sheet 前置拦截

| 项目 | 内容 |
|------|------|
| 改造点 | 消灭 AI 幻觉风险——所有事实信息必须来自用户输入 |
| 实施方案 | SKILL.md 新增"事实素材清单"节，按内容类型（探店/好物/教程/情感）定义必填字段 |
| 落地状态 | ✅ 已落地 — SKILL.md 第 31-68 行 |
| 验证方式 | 用户未填核心字段时，系统提示补全，不进入生成流程 |
| 责任角色 | R-01 PM 定义字段 + R-02 架构师实现 |

---

### 改造 #2（P0）：合规过滤器

| 项目 | 内容 |
|------|------|
| 改造点 | 钩子库含违规用语，输出无合规检查 |
| 实施方案 | hooks 拆分文件替换违规钩子 + SKILL.md 输出前合规过滤器（红线词/黄线词） |
| 落地状态 | ✅ 已落地 — hooks/5 文件已替换 + SKILL.md 第 147-159 行 |
| 验证方式 | 扫描全部钩子文件，无红线词；输出前自动扫描替换 |
| 责任角色 | R-06 法务提供词表 + R-02 架构师实现 |

---
